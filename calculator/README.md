## 🎯 Overview

This is a fully functional web-based calculator application built with HTML, CSS, and JavaScript. The application is containerized using Docker and served via Nginx web server.

**Purpose:** Learn Docker containerization, web deployment, and DevOps fundamentals.

---

## ✨ Features

- ✅ Basic arithmetic operations (Add, Subtract, Multiply, Divide)
- ✅ Responsive design (works on mobile & desktop)
- ✅ Clean, modern UI
- ✅ Dockerized for easy deployment
- ✅ Lightweight (< 10MB container)
- ✅ Fast loading with Nginx
- ✅ Cross-platform compatibility
